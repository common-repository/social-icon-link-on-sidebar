== Description ==

 This plugin easily make you able to add three different Social links and their icon on widget area.
According to this these three icon will be appeared on sidebar.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `social link Widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. GO to your widget Area and input social link and upload icon for them.
4.It will display on Sidebar.
